Reusing this in another app

  1. Copy the css/ folder. Link tokens → base → components → utilities; treat patterns as optional recipe-level CSS.
  2. Pick a theme: <html data-theme="light"> or leave default dark.
  3. Use the system accent by default, or brand it by overriding --accent and --accent-ink in your own stylesheet.
  4. Use the u-app scaffold for the titlebar / sidebar / status-bar shell.
  5. Compose screens from ds-* components. Nothing is app-specific.
