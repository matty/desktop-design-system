Context menu — interactive (with nesting)

Add data-ds-context="<id>" to any element. A hidden .ds-menu template with that id is cloned and positioned at the cursor on right-click. Nest a .ds-menu inside a .ds-menu-item.has-submenu for fly-out submenus.

Right-click anywhere in this card.
data-ds-context="ctxDemo" on the trigger · hidden #ctxDemo template cloned at cursor · .ds-menu-item.has-submenu with nested .ds-menu for fly-out · ArrowRight / ArrowLeft open and close submenus · Esc closes and returns focus
